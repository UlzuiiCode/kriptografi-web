ℹ️ Tentang Caesar Cipher & Mod 256

Pelajari cara kerja algoritma ini

Apa itu Caesar Cipher?

Caesar Cipher adalah salah satu teknik enkripsi tertua dan paling sederhana. Dinamakan dari Julius Caesar, yang menggunakannya untuk melindungi pesan militer. Cara kerjanya adalah dengan menggeser setiap karakter dalam pesan sebanyak posisi tertentu dalam alfabet (atau dalam kasus ini, tabel ASCII).

Bagaimana Implementasi Ini Bekerja?

Tool ini menggunakan tabel ASCII lengkap (0-255) bukan hanya huruf saja. Setiap karakter dikonversi ke nilai ASCII-nya, kemudian key ditambahkan (untuk enkripsi) atau dikurangi (untuk dekripsi), dan terakhir operasi modulo 256 diterapkan untuk menjaga hasil tetap dalam rentang ASCII yang valid.

Rumus Enkripsi:

ASCII Terenkripsi = (ASCII Asli + Key) mod 256

Rumus Dekripsi:

ASCII Terdekripsi = (ASCII Terenkripsi - Key + 256) mod 256

Apa itu Mod 256?

Operasi modulo (mod) mengembalikan sisa hasil pembagian. "Mod 256" memastikan bahwa hasilnya selalu berada dalam rentang 0-255, yang sesuai dengan set karakter ASCII extended. Ini menciptakan efek "wrap-around" (melingkar): jika penambahan key melebihi 255, maka akan kembali ke 0.

Contoh: Jika sebuah karakter memiliki nilai ASCII 250 dan key-nya 10, maka hasilnya adalah (250 + 10) mod 256 = 260 mod 256 = 4. Nilai tersebut melingkar kembali ke 4.

Catatan Keamanan:

Meskipun Caesar Cipher bagus untuk mempelajari konsep kriptografi, metode ini tidak aman untuk penggunaan di dunia nyata. Dengan hanya 256 kemungkinan key, enkripsi ini dapat dengan mudah dipecahkan dengan mencoba semua kemungkinan (brute force attack). Enkripsi modern menggunakan algoritma yang jauh lebih canggih.
